

































what is git ?
- Free & open source version control system 
- Tools to help to tracks changes in code
- Track history
- Help to collabrate

what is github ?
- Website where we host repositories online

how's work ?
- Create repository
- Add file
- Commit changes (stamp)

Using git :
- Command line
- IDE / Code editors (Like VSCode)
- Graphical user interface (Like GitKraken)

Configuring Git :
- git config --global user.name "My Name"
- git config --global user.email "someone@gmail.com"

show git configuring list :
- git config --list

Clone :
- Cloning a repository on our local machine
- git clone https://github.com/jay-sitapara/repo/self-study

Status :
- Display the state of the code
- git status
